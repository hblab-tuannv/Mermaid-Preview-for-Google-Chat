# MAIN-US-010 — Sửa preview lỗi với label `<br>` và sơ đồ C4 (parse SVG bằng text/html)

**Goal:** Mermaid sinh SVG có HTML-trong-`<foreignObject>` (thẻ void `<br>` cho label xuống dòng, và các box của sơ đồ C4). `parseSvg` đang parse bằng `image/svg+xml` (XML nghiêm ngặt) nên `<br>` chưa đóng làm XML hỏng → `parsererror` → marker lỗi. Đổi `parseSvg` sang parse `text/html` (khoan dung void tag, giữ đúng SVG namespace) để các sơ đồ này preview được, không phá vỡ đảm bảo no-exec/AC-2/AC-5. Nguồn: CR-MAIN-2026-06-12-04.
